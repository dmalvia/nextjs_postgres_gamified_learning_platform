1. Project Structure & Core Concepts
Authentication: Uses Clerk for user management.
Database: Drizzle ORM with schema for users, courses, lessons, enrollments, progress, achievements.
UI: Next.js App Router, shadcn/ui for components, Tailwind CSS (already set up).
Data Fetching: React Query.
2. API Routes (with Purpose)
API Route	Purpose
/api/achievements	Get all achievements and user’s earned status
/api/courses	List all courses with user’s enrollment status
/api/courses/[courseId]	Get course details and lessons for a course
/api/courses/[courseId]/enroll	Enroll the user in a course
/api/leaderboard	Get top users by points, user’s rank, streak, etc.
/api/progress	Track lesson completion, check and award achievements
/api/streak	Get and update user’s current and longest streak
/api/user/stats	Get user stats: level, XP, streak, completed courses, lessons, etc.
/api/user/sync	Sync Clerk user data with DB (create/update user profile on login)

User journey
1. New User Onboarding & Profile Sync
User visits the app and signs up/signs in via Clerk.
/api/user/sync is called to create or update the user profile in the database.
User is redirected to the dashboard.
2. Browsing & Enrolling in Courses
User lands on the dashboard and sees available courses (/api/courses).
User clicks a course to view details and lessons (/api/courses/[courseId]).
User clicks “Enroll” to join the course (/api/courses/[courseId]/enroll).
3. Learning & Progress Tracking
User views a lesson page, watches content, and marks it as complete.
Completion triggers /api/progress to update progress and check for achievements.
User’s streak and XP are updated (/api/streak, /api/user/stats).
4. Achievements & Gamification
As the user completes lessons/courses, achievements are awarded (/api/achievements).
User can view earned and available achievements on the Achievements page.
5. Leaderboard & Social Motivation
User visits the Leaderboard page to see their rank and compare with others (/api/leaderboard).
User’s current streak, points, and level are displayed.
6. Profile & Stats
User visits their profile or dashboard to see stats: level, XP, streak, completed courses, lessons, etc. (/api/user/stats).
7. Daily Return & Streaks
User returns daily, completes lessons, and maintains/increases their streak (/api/streak).
Streak and longest streak are updated and shown on the dashboard.
